📋 成績データを貼り付けて始める

KOAN の成績表か保存済み JSON を下に貼り付けると、卒業要件と GPA をブラウザ内で集計します。

状態 読み込み待機中
入力はブラウザ内だけで処理します。 反映後は「概要」タブに切り替わります。
JSON は上書きまたは追記で読み込めます。 状態を消したいときは「クリア」を使います。
  1. KOAN の「単位取得状況紹介」で表をコピーします。
  2. 「KOAN 貼付」タブに貼り付けて「KOAN を反映」を押します。
  3. 下の「概要」でカテゴリ進捗と不足要件を確認します。
  4. 必要なら「JSON 復元」タブから保存し、あとで復元します。
