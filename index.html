単位趣味レーター Web

KOAN の「単位取得状況紹介」を貼り付けるだけで、卒業要件と GPA をブラウザ内で集計します。 サーバー保存はしません。

1 KOAN をコピー 単位取得状況紹介をそのまま使います。
2 貼り付けて反映 置換と追記を切り替えられます。
3 要件と GPA を確認 不足要件と期別 GPA を即時表示します。
Engine
起動中

Python ロジックをブラウザ内に読み込んでいます。初回は数秒かかります。

Privacy

個人成績は保持しません

  • 初期状態では個人成績を表示しません。
  • 貼り付けたデータはブラウザ内だけで計算します。
  • 必要なら JSON として自分で保存できます。

入力

普段は KOAN 貼り付けだけで足ります。JSON は復元や共有用です。

最短ルート KOAN の表をヘッダー行ごと貼り付けて、そのまま反映してください。
復元用 以前に保存した JSON を貼り付けると、そのまま現在の状態に戻せます。
  1. KOAN の「単位取得状況紹介」で表を表示してコピーします。
  2. このページの `KOAN 貼付` に貼り付けて `KOAN を反映` を押します。
  3. `要件` タブで不足条件を確認し、`GPA` タブで期別推移を見ます。
  4. 必要なら `JSON` タブから保存して、あとで `JSON 復元` に戻します。
対応入力: KOAN のタブ区切り表
対応成績: S / A / B / C / F / 合 / 否
計算場所: ブラウザ内のみ
